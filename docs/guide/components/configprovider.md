## APIs

### ConfigProvider

参数 | 说明 | 类型 | 默认值
:-- | :-- | :-- | :--
theme | 主题对象 | [Theme](#theme-type) | {}
abstract | boolean | 是否不存在 `DOM` 包裹元素 | true
tag | string | `ConfigProvider` 被渲染成的元素，`abstract` 为 `true` 时有效 | 'div'

### Theme Type

名称 | 说明 | 类型 | 默认值
:-- | :-- | :-- | :--
common? | 全局通用配置，优先级低于组件配置 | [Config](#config-type) | undefined
