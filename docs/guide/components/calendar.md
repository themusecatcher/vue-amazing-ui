# 日历 Calendar

<GlobalElement />

*按照日历形式展示数据的容器*

## 何时使用

- 当数据是日期或按照日期划分时，例如日程、课表、价格日历等；目前支持年/月切换

## 参考文档

- [date-fns](https://date-fns.org/)

}

  console.log('panelChange', date, info, mode)
