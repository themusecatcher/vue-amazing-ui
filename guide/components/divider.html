分割线 Divider ​


区隔内容的分割线

何时使用 ​

  • 对不同章节的文本段落进行分割
  • 对行内文字/链接进行分割

基本使用 ​

Center Text
Center Text
Show Code
vue
<template>
  <Divider>Center Text</Divider>
  <Divider border-style="dashed">Center Text</Divider>
</template>

中间无文字 ​

Show Code
vue
<template>
  <Divider />
  <Divider border-style="dashed" />
</template>

Right Text
Show Code

  <Divider>Center Text</Divider>
  <Divider orientation="left">Left Text</Divider>
  <Divider orientation="right">Right Text</Divider>
</template>
